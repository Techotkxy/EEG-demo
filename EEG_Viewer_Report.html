OpenBCI EEG Viewer – Technical Report

Document: Cogwear Demo EEG Viewer
Version: Multi-headband real-time viewer with synchrony analysis

1. Overview

The EEG Viewer is a real-time visualization and analysis application for OpenBCI Cyton headbands. It supports 1–4 headbands simultaneously (4–16 channels), with preprocessing, frequency-band decomposition, and inter-subject EEG synchrony (Phase Locking Value) when multiple headbands are connected.

4. Display Windows

  • Raw EEG – Preprocessed signal (0.3–50 Hz bandpass, notch, ICA, z-score) in 5-second rolling window
  • Theta (4–8 Hz) – Band-filtered theta activity
  • Alpha (8–13 Hz) – Band-filtered alpha activity
  • Beta (13–30 Hz) – Band-filtered beta activity
  • Gamma (30–60 Hz) – Band-filtered gamma activity
  • EEG Synchrony (PLV) – Pairwise Phase Locking Value in alpha band (8–13 Hz); shown only when 2+ headbands are connected

Plots are arranged in a grid (2×2 for 4 channels, 2×4 for 8, 4×4 for 16). Packet loss and flat-signal indicators appear at the bottom of the Raw EEG window.

5. Signal Processing Pipeline

5.1 Preprocessing (applied to all channels)

StepSettingValue
BandpassLow / High cutoff0.3 Hz / 50 Hz (4th-order Butterworth)
NotchFrequency / Q50 Hz (or 60 Hz) / Q=30
Common Average Reference (CAR)Per headbandSubtract mean of 4 channels from each channel within that headband
ICAFastICA, kurtosis-based rejectionReject components with |z| > 1.96; refit every 2.5 s

5.2 Display-Only Processing

These steps affect visualization only, not stored or exported data:

  • Median filter (5-point) to reduce spikes
  • Outlier rejection: replace values beyond ±2 z-score with local median
  • Moving average smoothing (15-point)
  • Display clipping at ±2.5 z-score
  • Y-axis auto-range

6. EEG Synchrony (Multi-Headband)

When 2 or more headbands are connected:

  • Phase Locking Value (PLV) is computed in the alpha band (8–13 Hz) for all pairwise combinations (e.g. HB1–HB2, HB1–HB3, HB1–HB4, HB2–HB3, HB2–HB4, HB3–HB4 for 4 headbands).
  • Each headband’s signal is the mean of its 4 channels.
  • PLV ranges from 0 (no phase locking) to 1 (strong phase locking).
  • Bar chart updates in real time (~30 Hz).

7. Diagnostics and Monitoring

  • Packet loss – Per-headband count and percentage (e.g. HB1: 0.02% (7/33831))
  • Flat-signal indicator – “⚠flat” when a headband’s signal is much flatter than others (suggests poor electrode contact)
  • Cross-talk check – After ~5 s, warns if two headbands show correlation > 0.92 (possible same radio channel)
  • Radio channel reminder – Startup message when 2+ headbands; see RADIO_SETUP.md for configuration
